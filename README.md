# Cadastro de Cliente – Interface Responsiva e Validação de Dados

Este projeto é uma **página de cadastro de clientes** moderna e funcional, desenvolvida com **HTML5, CSS3 e JavaScript**. O objetivo principal é demonstrar o domínio de estruturas semânticas, estilização avançada com Flexbox e manipulação de eventos no front-end.

O projeto foi construído para compor meu portfólio e apresentar práticas sólidas de desenvolvimento para vagas de **Estágio em Desenvolvimento Fullstack**.

## ✨ Funcionalidades

* **Interface Semântica:** Uso de tags HTML5 para melhor acessibilidade e SEO.
* **Design Responsivo:** Layout que se adapta automaticamente a desktops e dispositivos móveis.
* **Validação de Formulário:** Lógica em JavaScript para capturar dados e validar campos obrigatórios.
* **Experiência do Usuário (UX):** Feedback visual com alertas de sucesso e limpeza automática do formulário.
* **Hospedagem Automática:** Configurado para demonstração online via GitHub Pages.

## 🛠️ Tecnologias Utilizadas

* **HTML5** – Estrutura semântica e formulários acessíveis.
* **CSS3** – Estilização moderna, variáveis e Media Queries para responsividade.
* **JavaScript (ES6)** – Manipulação do DOM e lógica de submissão de dados.
* **Git & GitHub** – Controle de versão e deploy.

## 📂 Estrutura do Projeto

```text
├── index.html   # Estrutura principal e formulário
├── style.css    # Design, cores e layout responsivo
└── script.js    # Lógica de captura e validação de dados
```

## 🎯 Objetivo do Projeto

Este projeto demonstra competências fundamentais para um desenvolvedor em início de carreira:
1.  **Fundamentos de Front-end:** Escrita de código limpo e organizado.
2.  **Acessibilidade:** Uso correto de labels e tipos de input.
3.  **Lógica de Programação:** Tratamento de eventos de formulário com JS.
4.  **DevOps Inicial:** Fluxo completo de Git (add, commit, push) e publicação em nuvem.
